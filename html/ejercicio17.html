Términos Comunes en el Desarrollo Web

HTML (HyperText Markup Language)
El lenguaje estándar de marcado para crear y estructurar páginas web.
CSS (Cascading Style Sheets)
Un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML.
JavaScript
Un lenguaje de programación interpretado que se utiliza para crear contenido dinámico en sitios web.
API (Application Programming Interface)
Un conjunto de reglas que permite que las aplicaciones se comuniquen entre sí.
Frontend
La parte de un sitio web que interactúa directamente con el usuario y está compuesta por HTML, CSS y JavaScript.
Backend
La parte del sitio web que maneja la lógica de negocio, la base de datos y las interacciones con el servidor.
dt>Framework
Una plataforma de software que proporciona una base para desarrollar aplicaciones de manera más eficiente.
Responsive Design
Un enfoque de diseño web que permite que las páginas web se adapten a diferentes tamaños de pantalla y dispositivos.
SEO (Search Engine Optimization)
El proceso de mejorar la visibilidad de un sitio web en los resultados de búsqueda de los motores de búsqueda.
Git
Un sistema de control de versiones distribuido que permite a los desarrolladores rastrear cambios en el código fuente durante el desarrollo del software.
